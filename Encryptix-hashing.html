Hashing (Vitalina)

This demo shows how a system can store only a hash (fingerprint) of a password and later check if a new password matches, without ever seeing the original text.

No hash saved yet. Type text in Step 1 and press “Save hash”.

Hashing turns your text into a one-way fingerprint. The system stores only this fingerprint. When you type your text again, it is hashed once more and the fingerprints are compared. If they match, the text is the same; if not, even a tiny change gives a different hash.
