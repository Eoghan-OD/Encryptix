Case Studies

Symmetric Encryption in a Veterinary Clinic

A mid-sized veterinary clinic stored X-rays, blood test results and medical histories on an unencrypted external drive, creating a GDPR Article 32 compliance issue.

Encryptix implemented AES-256 encryption through BitLocker, generated secure recovery keys and trained staff on proper key management. The clinic now has fully encrypted data at rest with no disruption to workflow.

Asymmetric Encryption for Secure Referrals

A veterinary practice shared sensitive referral documents with a specialist hospital using unencrypted email, leaving communication vulnerable to interception.

Encryptix deployed TLS 1.3 using OpenSSL, created RSA key pairs and enforced encrypted transmission for all outgoing referrals. Communications are now secure, private and GDPR compliant.

Hashing to Protect Staff Credentials

An animal shelter stored staff passwords in plain text within a shared spreadsheet, posing an immediate security risk.

Encryptix replaced this with SHA-256 hashing and unique salts, removing all readable passwords from the system and greatly improving security resilience.

Hybrid Encryption for Secure Data Backups

A regional animal shelter stored weekly database backups containing adoption records, vaccination details and payment information on a cloud service without any form of encryption.

Encryptix implemented a hybrid encryption system using AES-256 for fast data encryption and RSA-2048 for secure key exchange. Automated scripts were set up to encrypt each backup before upload, ensuring both speed and strong protection. The shelter now benefits from fully encrypted, tamper-resistant cloud backups.
