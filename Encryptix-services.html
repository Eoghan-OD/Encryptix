Our Encryption Demonstrations

The Encryptix team created these interactive demos to help veterinary clinics see how different types of encryption work in practice. Each demonstration represents one of the key techniques used to secure information in modern systems.

Symmetric Encryption (Sean)

Uses one shared key to both encrypt and decrypt information. This demo shows a simple Caesar cipher in action.

Asymmetric Encryption (Eoghan)

Uses a public key for encryption and a private key for decryption. This demo shows the concept of RSA-style key exchange.

Hashing (Bradley)

Hashing converts text into a fixed-length, irreversible value. It is used to protect passwords and verify data integrity.

Hybrid Encryption (Vitalina)

Combines symmetric and asymmetric encryption. The message is encrypted with a symmetric key, which is then protected using an asymmetric key.
